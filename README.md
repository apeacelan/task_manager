# task_manager
Flask app for managing your tasks with prioritizing and beautiful design

**Task manager flask web application deployment guide **

 
Prerequisites  

System Requirements 

Windows 10/11 (64-bit recommended) 

Python 3.8 or higher 

Git (optional) 

 

Software installation  

Install Python: 

1. Download from python.org 

2. During installation add python to PATH 

3. Verify installation  

python --version 

pip –version 

 

 Install Git (optional): 

winget install Git.Git 

 

Application start  

# 1. Clone repository 

git clone https://github.com/apeacelan/task_manager.git 

cd task_manager 

 

# 2. Create virtual environment 

python -m venv venv 

 

# 3. Activate it 

venv\Scripts\activate 

 

# 4. Install dependencies 

pip install –r requirements.txt 

 

# 5. Run application 

python app.py 

 

Access on localhost
